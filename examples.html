Kaan.js

a 2D physics simulation library for JavaScript

click here for documentation

click here for GitHub repo

Click start button to start the enviroment
With this library, you can generate an enviroment and in that enviroment, you can create solid, liquid, and gas materials
Enviroment has parameters such as gravity and color, some objects respond to some of these variables
Start! button envokes the code: env1.startEnviroment()
Code:

Matters can be initialized in a way to let end user interact with objects by clicking and dragging them.
If user drops the object gravity will be applied to the object.
Code:
            <script>
            const env2 = new Enviroment("env2", 300, 600, '#003366', 0, 200)
            env2.createEnviroment()
            const env2_solid1 = new Matter(env2, 40, 60, 200, 150, 'Orange', 100, true, true, "orange")
            env2_solid1.createSolid()
            const env2_solid2 = new Matter(env2, 90, 30, 120, 100, 'Cyan', 20, true, true, "cyan")
            env2_solid2.createSolid()
            const env2_solid3 = new Matter(env2, 30, 80, 130, 200, 'Purple', 20, true, "purple")
            env2_solid3.createSolid()
            env2.startEnviroment()
            </script>
Matters can be made reactable
Oxygen! button envokes the code: oxygen.showItem()
Hydrogen! button envokes the code: hydrogen.showItem()
Start! button envokes the code: env3.startEnviroment()
Code:
            <script>
            const env3 = new Enviroment("env3", 300, 600, '#eeeeee', 10, 200)
            env3.createEnviroment()
            const oxygen = new Matter(env3, 40, 40, 180, 150, '#afdfff', 100, false, false, "oxygen")
            oxygen.createGas()
            const hydrogen = new Matter(env3, 40, 40, 380, 150, '#afefef', 100, false, false, "hydrogen")
            hydrogen.createGas()
            const water = new Matter(env3, 40, 40, 380, 150, 'Blue', 100, false, false, "water")
            water.createLiquid()
            oxygen.setReactable(hydrogen, water)
            </script>
Matters will react to changes in enviroment.
The slider envokes env5.changeTemparature(this.value) on change.
Change Temparature
-200 200
Code:
            <script>
            const env5 = new Enviroment("env5", 300, 600, "lightgreen", 0, 200);
            env5.createEnviroment();
            const env5_item1 = new Matter(env5,200,40,200.1,100,"Orange",100,true,true,"sand1",-100,100);
            env5_item1.createSolid();
            const env5_item2 = new Matter(env5,200,40,200.1,100,"Blue",100,true,true,"blue",-200,200);
            env5_item2.createSolid();
            env5.startEnviroment();
            </script>
Library lets developers attach a user interface to the enviroments to add new Matters
Code:
            <script>
            const env4 = new Enviroment("env4", 300, 600, 'Black', 0, 200)
            env4.createEnviroment()
            const env4_solid1 = new Matter(env4, 50, 50, 220, 100, 'Red', 100, true, true, "red")
            env4_solid1.createSolid()
            const env4_solid2 = new Matter(env4, 80, 40, 300, 150, 'Green', 20, true, true, "green")
            env4_solid2.createSolid()
            const env4_solid3 = new Matter(env4, 100, 20, 230, 250, 'Blue', 20, true, true)
            env4_solid3.createSolid()
            env4.startEnviroment()
            new EnviromentController(env4).create()
            </script>
